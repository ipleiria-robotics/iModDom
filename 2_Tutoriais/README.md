    4.[Tutorial_Utilização_dos_Blocos](https://github.com/MarcoPereira00/aiModDom/blob/master/iModDom/Tutoriais/Tutorial_Utiliza%C3%A7%C3%A3o_dos_Blocos.docx "Tutorial_Utilização_dos_Blocos")
